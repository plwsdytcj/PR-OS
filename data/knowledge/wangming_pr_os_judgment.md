# 王明公关传媒营销品牌行业 OS · PR-OS 判准摘要

## 结算系统
客户现实 + 预算 + 产品事实 + 品牌语义 + 媒体/达人节点 + 传播环境
→ 选题与渠道匹配 → 交付 → 证据 → 付款 → 案例回写 → 可复用资产。

## 六种业务类型
- 公关：可审计链接、叙事站位、关系方态度、风险可控
- 营销：活动产出、有效触达、话题参与、可量化反馈
- 品牌：长期语义沉淀、识别度、信任与溢价
- 效果：ROI、线索、转化、订单等量化指标
- 商誉：投资人/合作方信心、第三方背书、预期管理
- 运营：激活、留存、复购、社群参与

## 二段传播
优先高信任媒体/达人节点转译信息，而非纯流量曝光。适合需要解释、背书、专业转译的场景。

## 八条流
资源流、成本流、权限流、符号叙事流、情绪压力流、劳动流、反馈流、退出流。

## 红线
不做假量、未披露商单、攻击抹黑、非法监测、虚构数据。无验收标准、无付款节点、无退出规则不推进。

## 项目结案
发布链接、截图、数据、客户确认、付款记录、案例回写、达人卡更新、风险备注。
